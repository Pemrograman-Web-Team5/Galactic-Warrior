About Us

Galactic Warrior

This game challenges players to take control of a attacker in an epic battle against enemies attacks. By firing strategically, the player must avoid enemy attacks and survive as long as possible. Each time the attacker is hit by an enemies attack, the game will end, and the player will lose. Brave the challenge and survive as far as possible in this game!

Flowchart

The flowchart flow in this game starts from the start. Then the user must click the space bar on the keybord to launch fire on the rocket at the alien. Then click the arrow keys on the keyboard to move the rocket right and left. Then if you choose pause, the game stops. At the game stop there is a choice of play or not if you choose play then the game will continue. If not, it will loop until the user chooses to play. If you don't pause, there is another choice, namely whether you managed to shoot the target? if so, it will add 100 points and the flow will return to the top. If not, it will be game over.

How To Play

Point The Enemies

When the enemies moves from the right side to the left side, we can control the attacker to shoot the enemies by pressing the space bar.

Shoot The Enemies

When we shoot enemies, enemies will reappear with an increasing number, we must fend off enemies by firing bullets with attacker.

Avoid Enemies

If the enemies have moved downwards successfully approaching the attacker, the attacker can no longer move and the game is over.
